
# 🤖 AI Cannel Bot

Автоматический Telegram-бот для публикации IT-новостей с генерацией текста и изображений через ИИ.

**Статус:** Активная разработка 🔥 • Стабильность значительно улучшена

Наш канал: [t.me/ai_cannel](https://t.me/ai_cannel)

---

## ✨ Возможности

- 📰 Парсинг свежих новостей из 9+ популярных IT-источников (Habr, 3DNews, vc.ru, iXBT, SecurityLab и др.)
- ✍️ Живая генерация постов с помощью LLM (Twelver, Groq, Hugging Face и ещё 4 провайдера с автоматическим fallback)
- 🖼 Автоматический подбор обложки: сначала Pexels, затем генерация через ИИ
- 💬 Умные ответы на комментарии в обсуждении с помощью ИИ
- 🧹 Автоматическое удаление дубликатов постов в группе общения
- 🎲 Интерактивные квизы по новостям (с заданной вероятностью)
- 🛡️ Защита от дубликатов новостей через SQLite
- ⏰ Автозапуск каждый час через GitHub Actions
- 🧪 Тестовый режим + команды для создателя (`/generate`, `/stats`)
- 📋 Подробное логирование и улучшенная обработка ошибок
- 🌐 Приоритет свежих новостей (возраст, ключевые слова)

## ⚙️ Настройка

### 1. Создай бота
- Напиши `@BotFather` → создай нового бота → получи `TG_BOT_TOKEN`
- Добавь бота **администратором** в свой канал
- Добавь бота **администратором** в группу обсуждения (если используешь комментарии)

### 2. GitHub Secrets

Перейди в **Repository → Settings → Secrets and variables → Actions** и добавь:

| Секрет                | Описание                                | Обязательно    |
|-----------------------|-----------------------------------------|----------------|
| `HF_API_TOKEN`        | Токен от Hugging Face                   | Да             |
| `TG_BOT_TOKEN`        | Токен твоего Telegram-бота              | Да             |
| `TG_CHAT_ID`          | ID канала (`-1001234567890`)            | Да             |
| `TG_GROUP_ID`         | ID группы обсуждения                    | Рекомендуется  |
| `CREATOR_ID`          | Твой Telegram ID                        | Да             |
| `PEXELS_API_KEY`      | API ключ от Pexels                      | Желательно     |
| `GROQ_API_KEY`        | API ключ от Groq                        | Опционально    |
| `ALGION_API_KEY`      | API ключ от Algion                      | Опционально    |
| `TWELVER_API_KEY`     | API ключ от Twelver                     | Опционально    |

### 3. Запуск

**Локально:**
```bash
python bot.py
```

**Тестовый режим (рекомендуется для проверки):**
```bash
TEST_MODE=true python bot.py
```

**Через GitHub Actions** — бот запускается автоматически каждый час.

## 🔄 Как это работает

1. Парсит RSS-фиды популярных IT-сайтов
2. Отбирает самые свежие и интересные новости (возраст, приоритет источников, ключевые слова)
3. Генерирует живой пост с помощью ИИ (перебор провайдеров)
4. Подбирает или генерирует обложку
5. Публикует в канал
6. Сохраняет всё в базу
7. С некоторой вероятностью создаёт квиз по другой новости
8. При следующих запусках проверяет комментарии, отвечает на них, удаляет дубликаты

**Время публикации:** 9:00, 10:00, 12:00, 14:00, 16:00, 18:00 по Ижевску ⏰

## ⚠️ Текущие известные проблемы

- Иногда Hugging Face выдаёт 429 (rate limit) — бот это обрабатывает, но пост может пропуститься
- При долгом отсутствии комментариев offset может немного залипать (решается ручным перезапуском)
- Генерация изображений через AI пока не всегда стабильна

Я активно работаю над улучшением стабильности!
